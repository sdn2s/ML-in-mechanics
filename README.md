# ML-in-mechanics

Решения ноутбуков курса.

## Как запустить
- Создайте виртуальное окружение: `python3 -m venv .venv && source .venv/bin/activate`.
- Установите зависимости: `pip install -r requirements.txt`.
- Запустите Jupyter: `jupyter notebook` и выберите интерпретатор из `.venv`.
- Для корректной работы в IDE можете добавить ядро: `python -m ipykernel install --user --name ml-in-mechanics --display-name "Python (.venv)"`.
